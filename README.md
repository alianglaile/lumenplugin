# Lumen Player 插件倉庫

[📖 English Version](README_EN.md)

本目錄包含 Lumen Player 的視頻源插件。每個插件為一個獨立子目錄，透過 `config.json` 定義頁面結構與 API 映射，搭配 JavaScript 腳本解析第三方網站內容。

## 🎬 插件詳情

### `douban/` — 豆瓣精選

豆瓣電影資料庫的主題型插件，提供精選推薦與排行榜瀏覽，**不提供播放功能**。適合作為主題源（theme）搭配其他視頻源使用——在豆瓣瀏覽到感興趣的影片後，Lumen 會自動從其他視頻源搜尋播放連結。

**支援頁面：**

| 分類 | 說明 |
|------|------|
| 主頁 | 聚合熱門電影、熱門劇集、熱門綜藝 |
| 熱門電影 / 最新電影 | 依時間與熱度排序 |
| 豆瓣高分 / 冷門佳片 | 依評分篩選 |
| 豆瓣新片榜 / 一周口碑榜 / 北美票房榜 | 各類排行榜 |
| 豆瓣 Top250 | 經典佳作榜單 |
| 年度榜單（2021–2025） | 歷年年度精選 |

**檔案結構：**
- `config.json` — 頁面定義與 API 端點
- `main.min.js` — 資料解析邏輯（已壓縮）

### `olevod/` — 新歐樂影院

綜合影視串流插件，涵蓋電影、連續劇、短劇、綜藝、動漫等分類。支援完整的瀏覽、搜尋與播放功能。

> ⚠️ **阻擋中國大陸 IP**，適合海外使用者。

**支援頁面：**

| 分類 | 說明 |
|------|------|
| 首頁 | 最新上架內容 |
| 電影 | 院線及經典電影 |
| 連續劇 | 陸劇、美劇、韓劇等 |
| 短劇 | 短篇劇集 |
| 綜藝 | 各類綜藝節目 |
| 動漫 | 動畫及漫改作品 |

**技術特點：**
- 使用 `crypto-js` 進行 API 簽名認證
- 支援首頁 Hero Banner 輪播
- 支援關鍵字搜尋

**檔案結構：**
- `config.json` — 頁面定義與 API 端點
- `main.js` — 主要資料解析與簽名邏輯
- `crypto-js.min.js` — 加密函式庫
- `t.js` — 輔助工具腳本

### `jable/` — Jable.TV（18+）

成人影片串流插件，提供高清 1080P 內容。支援多種分類瀏覽、搜尋與直接播放。

**支援頁面：**

| 分類 | 說明 |
|------|------|
| 主頁 | 聚合最近更新、本周熱門等 |
| 最近更新 / 本周熱門 | 依時間與熱度排序 |
| 中文字幕 / 無碼解放 | 依字幕或類型篩選 |
| 其他分類 | 制服誘惑、角色劇情、女同歡愉等 |

**檔案結構：**
- `config.json` — 頁面定義與搜尋 URL 模板
- `main.js` — HTML 頁面解析邏輯（非壓縮，可直接閱讀）

## 🔧 插件目錄結構

每個插件遵循統一的檔案結構：

```
<plugin-name>/
├── config.json       # 插件配置（頁面、搜尋、播放、劇集的 URL 與 JS 函數映射）
├── main.js           # 主要 JavaScript 邏輯
└── *.js              # 可選的輔助檔案（如 crypto-js.min.js）
```

### `config.json` 核心欄位

| 欄位 | 說明 |
|------|------|
| `name` / `description` | 插件名稱與描述 |
| `host` | 目標網站主域名 |
| `files` | 需載入的 JS 檔案列表（按順序載入） |
| `pages` | 頁面定義陣列，每頁指定 `key`、`title`、`url`、`javascript` 函數名 |
| `search` | 搜尋功能配置（URL 模板 + JS 函數） |
| `episodes` | 劇集列表取得配置 |
| `player` | 播放 URL 解析配置 |
| `heroBanner` | 首頁輪播橫幅配置（可選） |

## 🚀 如何在 Lumen Player 中使用

1. 插件透過 GitHub 託管，以 Raw URL 提供存取
2. 在視頻源配置（`sources.json`）中，將 `api` 欄位設為：
   ```
   lumen://https://raw.githubusercontent.com/alianglaile/lumenplugin/main/<plugin-name>/config.json
   ```
3. Lumen Player 會自動下載 `config.json` 及其引用的 JS 檔案並載入執行
